Calculus Series Guide

Language: HTML/CSS
Environment: Webflow
Role: Solo Developer

A modern math tutorial website

HTML/CSS
Webflow

2021

My Calculus Series Guide aims to explain the process for solving a particular math problem. The project was done under the prompt of a one-page website that creates a narrative. With my background as a tutor, I wanted to create a website that involves solving a series problem. Solving for the convergence of a series requires a sequential application of tests to determine the answer, which fits a one-page design quite well.

Very often, math education websites value functionality over the appearance of the website. I tried to combat this trend by creating both a functional and visually appealing tutorial website. This website was created using webflow, which, at a basic level, provides a visual editor for HTML and CSS properties. This project involved creating wireframes, design mockups, and eventually, the final website you see here.
